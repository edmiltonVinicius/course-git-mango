- Untrucked File
    Arquivos Novos que o git não conhece


• git restore --staged nome-do-arquivo  
    Retorna o arquivo para a sessão unstaged area (o arquivo não será enviado no próximo commit)


• No arquivo de configuração, seja do sistema, usuário ou projeto, podemos criar alias atalhos
    [alias]
        atalho = !comando-do-atalho  (a ! é necessário para que o comando seja executado)
        • ex: s = !git status
            Depois no terminal rode git atalho  
                • git s

        • ex 2: c = !git add . && git commit -m
             Depois no terminal rode git atalho  
                • git c "texto do commit"


• git commit --amend --no-edit  
    Caso vc queria adicionar mais alterações no ultimo commit sem mudar a mensagem do ultimo commit


• git stash
    Guarda as alterações não commitadas na memória stash, para podermos obte-las depois    

• git stash list
    Lista todos os stash

• git stash apply id
    Adiciona as alterações em stash passando o id *** O APPLY ADICIONA O STASH MAS MANTEM O STASH

• git stash pop id
    Adiciona as alterações em stash passando o id *** O POP ADICIONA O STASH E REMOVE DA LISTA

• git stash clear
    Apaga todos os stash
